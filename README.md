# Prepview AI ✨
Your Personal AI Interview Coach. Practice, get feedback, and land your dream job.

PrepViewAI is an intelligent interview preparation platform that leverages cutting-edge AI to provide a realistic, conversational interview experience. Our advanced AI voice agent doesn't just ask questions—it listens, understands, and provides instant, actionable feedback to help you improve.

## 🛠️ Tech Stack

**Core technologies:**
- 🔷 Next.js   
- 🔒 Firebase (Auth, Firestore, Storage)  
- 🎨 Tailwind CSS + shadcn/ui  
- 🗣️ Vapi  
- 🧠 Google Gemini  
- ✅ Zod  
- 🚀 Vercel (deployment)  

---

## ☸ Feature Breakdown
- 🔐 **Authentication**: Firebase email/password auth  
- 🎙️ **AI-driven Interviews**: Real-time conversations with Vapi voice agents  
- 🧾 **Transcripts**: Auto-generated with feedback from Gemini  
- 📊 **AI Feedback**: Strengths, weaknesses, and follow-up suggestions  
- 🗂️ **Dashboard**: Manage past interviews and scores  
- 📱 **Responsive UI**: Built with Tailwind + shadcn/ui  
- ♻️ **Reusable Architecture**: Route groups, nested layouts, component-driven
